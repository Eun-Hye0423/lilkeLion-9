# HTML 기초

## HTML
-> 컨텐츠의 구조를 정의하는 마크업 언어
<br>
요소
-> 시작태그, 내용, 종료태그

<br>

    !DOCTYPE html 
= 코드x  ,문서형식을 정의

    html lang = "kr"
 = 본격적인 태그의 시작, 사용하는 주 언어를 정의

<br><br>

## Head와 Body에 들어갈 요소
<br>
    meta charset="utf-8"
= 문서와 관련된 정보를 담음

    title
웹 페이지의 제목을 담음

<br>

## 레이아웃과 관련된 기본태그

시맨틱 태그 =의미를 가지고 있는 태그

* header
* nav
* section
* article (주 내용을 담기위해 사용되는 요소)
 
* aside  (광고나 사이트의 주변부분에 해당하는 내용을 담기위해 사용되는 요소)
  
* footer (추가정보를 담기위해 사용하는 요소)
