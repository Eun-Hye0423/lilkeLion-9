# 터미널사용법2

## 외워야 할 명령어

* pwd
* man
* ls
* cd
* clear
  

## 명령어
명령어 [옵션][인자...]

* 옵션 

= "_"로 시작해서 영문 대소문자를 구성 명령어의 기능을 구체화 명령어에 따라 없을 수 도 있다.

* 옵션 사용법
  ls-l, ls-a, ls-al, ls-la


* 인자

= 명령어의 수행시 대상이 될 파일이나 디렉터리 명령어에 따라 필요 없을 수도 있고 필수일 수도 있다

* pwd

=현재위치를 알려준다

* man
  
= 명령어 설명서

man[알고싶은 명령어]

* ls(list)

= 디렉터리의 목록을 보여준다.

a -> 숨김파일까지 보여줌

l -> 상세하게 보여줌

F -> 파일인지 디렉터리인지 알려줌

* cd

= 현재 위치를 이동해줌
cd[이동하고 싶은 위치]

* clear
  
= 터미널 청소기
